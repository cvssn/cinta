# contribuindo com o cinta

agradecemos o seu interesse no **cinta**. o cinta é um projeto de pesquisa com foco acadêmico

nossos princípios fundamentais são **minimalismo de código**, **alta legibilidade** e **pureza funcional**

## 1. contribuições preferidas

- **correção de bugs e novas features**
  esses são nossos principais focos

- **minimalismo antes**
  o código deve ser conciso, claro e minimalista.

## 2. padrões rejeitados

- **engenharia excessiva**
  rejeitaremos encapsulamento desnecessário, abstração excessiva ou refatoração arquitetural complexa.
  lembre-se que este código tem fins de pesquisa e não se trata de um projeto empresarial comercial

- **ajustes de estilo**
  solicitações de pull request (prs) que visem exclusivamente formatação, embelezamento ou ajustes de estilo não funcionais serão rejeitadas

- **contribuições em outros idiomas**
  solicitações de pull request (prs) contendo comentários de código, documentação, mensagens de commit ou descrições em idiomas diferentes do inglês serão rejeitadas

## 3. política de review

- **análise linha por linha**
  os responsáveis ​​pela manutenção auditam rigorosamente cada linha de alteração manualmente. Antes de submeter o código, certifique-se de ter analisado pessoalmente e compreendido completamente toda a lógica do código

- **atenção em relação ao código gerado por ia**
  o código gerado pelo llm frequentemente contém lógica redundante e defeitos sutis. grandes trechos de código gerado por ia serão rejeitados, a menos que tenham passado por uma rigorosa limpeza e verificação humana.
  garanta que cada linha de código tenha uma necessidade absoluta de existir

## 4. normas de documentação

- **preciso e conciso**
  a documentação do cinta serve a uma comunidade de pesquisa global. mantemos padrões extremamente elevados: as alterações devem ser sucintas e precisas.
  elimine toda a verbosidade e redundância para garantir a máxima densidade de informação

---

aguardamos com expectativa contribuições de pesquisa **precisas, concisas e substancialmente valiosas**
